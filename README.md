# YOLOv10-Analysis-and-identification-of-3D-printing-defects
最近几年，3D打印技术快速发展，逐步走进人们的日常生活，其中，基于FDM技术的3D打印由于成本低廉，操作简单，易于制作，系统安全，个性定制等等优点，被广泛应用于教育、医疗、军事、航空航天等多个领域。得益于高速打印技术的进一步开发和愈加完善的3D打印工厂，打印产品的生产速度进一步提高，但更快的生产也暴露出了FDM3D打印在生产成型过程中容易出现各种缺陷的问题。

本文针对解决FDM3D打印过程出现的缺陷，提出了利用机器视觉结合深度学习神经网络的检测方法。通过对打印缺陷进行分析总结，收集打印缺陷的图像数据，经过处理后交给建立好的模型进行训练，不断调整参数，提高模型性能。最后在模型的基础上，创新设计了显示缺陷数据的GUI界面，方便使用者判断缺陷的各项数据，还可直接保存缺陷数据，使用简便快捷，可以极大提高缺陷检测的效率和准确率。若将其投入实际应用进一步普及，还能提高检测准确率和检测范围。
